Buen día, les paso los pasos a seguir para subir las imágenes al github:

Paso 1: git clone o git pull https://github.com/vanedaza/piscis.git, 
Paso 1a: Por las dudas hacer una copia del repo, sin hacer nuestra modificaciones.

Paso 2: En la carpeta hay un archivo oculto de git llamado  '.gitignore' (ver con ls -a) en el mismo hay que o verificar que
 esté comentada la linea 57 (#db.sqlite3).

Paso 3: Hacer python manage.py runserver

Paso 4: Ingresar a la dirección http://127.0.0.1:8000/admin, osea agregar el /admin  

Paso 5:  Ingresar el nombre de usuario: piscis y la contraseña: piscis2020

Paso 6: Cargar las imágenes que le corresponden (Grabar y añadir otro) en orden

Paso 7: Si quieren ver si todo va bn pueden usar DB Broser for SQLite para ver la tabla correspondiente a las imágenes. 
Con este software puede manipular las tablas como quieran.

Paso 8: Cerrar el entorno virtual y hacer
        git add .
        git commit -m 'imagenes 0-125'
        git push
